## 使用方法

```
// 铅笔动画
[_contentView showEWTLoadingWithTitle:@""];

// 铅笔动画 + 文字
[_contentView showEWTLoadingWithTitle:@"正在加载中..."];

// 菊花转圈加载效果
[MBProgressHUD showLoadingToView:_contentView];

```
